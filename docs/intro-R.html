Podemos almacenar muchos tipos de objetos en variables:

Para ver los elementos en objetos como vectores podemos usar √≠ndices:

## [1] "a" "e"
# Creemos un nuevo vector num√©rico

# Ver solamente elementos pares

## [1] 2 4 6
